💻 Reflection: Coding CV using Markdown (STAR Method)
Situation

As part of the digital portfolio assignment, I was required to create and present my CV using Markdown on GitHub. This was a new experience for me as I had not previously used Markdown to structure professional documents.

Task

My task was to convert a traditional CV into a well-structured digital format using Markdown syntax while ensuring that the information remained clear, professional, and easy to read.

Action

To complete this task, I researched how Markdown works and learned how to use headings, bullet points, and formatting techniques to organise my content. I carefully structured my CV into sections such as personal information, skills, and education. I ensured consistency in formatting and focused on readability so that anyone viewing the CV could easily understand my qualifications and abilities.

Result

As a result, I successfully created a clean, well-organised digital CV that is accessible online. This process improved my technical skills in Markdown and enhanced my ability to present information professionally in a digital format.
